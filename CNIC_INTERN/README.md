# README —— CAS - CNIC cyber 实习科研项目总结

## 一、项目1：网页截图水印算法
### 1. 项目背景
互联网网页截图易被篡改、无来源标识，传播后无法追溯原始网页归属，针对网页截图可信溯源需求，设计**基于网页关键特征的不可见水印方案**。

### 2. 核心思路
1. 自动化抓取目标网页，解析页面文本、标题、链接、特征文本等关键核心内容；
2. 对提取的网页特征进行编码压缩，生成轻量化水印信息；
3. 提取流程：任意截图输入算法，反向解码恢复原始网页关键信息，完成截图来源核验。

## 二、项目2：基于前缀树(Trie)的网页溯源检索系统（配套论文2）
### 1. 项目背景
海量网页数据场景下，传统字符串匹配检索效率低下，难以快速完成网页来源匹配、跨页面溯源校验，基于前缀树优化网页域名、URL、文本检索流程。

### 2. 核心思路
1. 爬取、清洗全网/指定站点网页URL、域名、关键标识文本；
2. 使用Trie前缀树构建多层索引，分层存储域名分段、URL路径、页面特征词；
3. 输入待核验网页片段/短URL，在前缀树中快速前缀匹配，批量召回同源网页；
4. 匹配结果打分，输出可信度最高的原始来源链接，实现网页溯源。
