
# 这里的几个程序是用来**批量生成模拟财务报表表格图片**的工具，主要用于生成带有财务数据的表格截图。

---

## 文件概览

| 文件名 | 作用 |
|--------|------|
| `normal.py` | 生成**标准财务表格**（带下划线、交替行颜色、分组表头） |
| `financial_report.py` | 生成**简化版财务表格**（带分组表头、下划线） |
| `noborders.py` | 生成**纯无线条表格**（完全无边框、无颜色） |
| `noborders_nocolors.py` | 生成**纯无线条但有颜色**的表格（表头深蓝、行标签浅蓝） |
| `requirements.txt` | 依赖包列表 |

---

## 各文件详细说明

### 1. `normal.py` - 标准财务表格

```
┌─────────────────────────────────────────────┐
│  For the Year Ended December 31, 2024       │
│  (in USD thousands)                         │
├──────────┬──────────┬──────────┬───────────┤
│          │ 2024 $   │ 2023 $   │  2022 $   │
├──────────┼──────────┼──────────┼───────────┤
│ Revenue  │ 1,234,567│ 1,123,456│ 1,012,345 │  ← 交替行颜色
│ COGS     │  234,567 │  212,345 │  198,765 │
├──────────┼──────────┼──────────┼───────────┤
│ Gross    │ 1,000,000│  911,111 │  813,580 │
│ Profit   │   ───    │   ───   │   ───    │
╞══════════╪══════════╪══════════╪═══════════╡
│ Net Inc  │  500,000 │  450,000 │  400,000 │  ← 双下划线
└──────────┴──────────┴──────────┴───────────┘
```

**特点**：
- ✅ 交替行颜色（浅蓝/白）
- ✅ 单下划线、双下划线
- ✅ 分组表头（如 "Revenue" 覆盖多列）
- ✅ 1000 张批量生成
- ✅ 同时输出 PNG + HTML

---

### 2. `financial_report.py` - 简化版表格

```
┌─────────────────────────────────────────────┐
│  Consolidated Financial Data                │
│  (USD in millions)                         │
├──────────┬──────────┬──────────┬───────────┤
│          │ Revenue  │  Cost    │   Margin  │  ← 分组表头
│          │ ($M)     │  ($M)    │   %       │
├──────────┼──────────┼──────────┼───────────┤
│ Net Sales│ $50,000  │ $30,000  │   40%     │
│ COGS     │ $20,000  │ $15,000  │   25%     │
├──────────┼──────────┼──────────┼───────────┤
│ Gross Prf│ $30,000  │ $15,000  │   50%     │  ← 下划线
├──────────┼──────────┼──────────┼───────────┤
│ Net Inc  │ $10,000  │  $5,000  │   20%     │  ← 下划线
└──────────┴──────────┴──────────┴───────────┘
```

**特点**：
- ✅ 分组表头（colspan 合并）
- ✅ 带缩进的行标签
- ✅ 单/双下划线
- ✅ 100 张批量生成
- ✅ 同时输出 PNG + HTML

---

### 3. `noborders.py` - 纯无线条表格（极简）

```
Consolidated Financial Data
(USD in millions)

           Revenue($M)  Cost($M)  Margin%
Net Sales    $50,000   $30,000     40%
  COGS       $20,000   $15,000     25%
Gross Profit $30,000   $15,000     50%
Net Income   $10,000   $5,000      20%
```

**特点**：
- ✅ 完全无边框
- ✅ 无任何线条
- ✅ 纯文字表格
- ✅ 100 张批量生成
- ✅ 同时输出 PNG + HTML

---

### 4. `noborders_nocolors.py` - 纯无线条 + 颜色

```
┌──────────┬──────────┬──────────┬───────────┐
│          │ Revenue  │  Cost    │  Margin   │  ← 深蓝背景 白色文字
├──────────┼──────────┼──────────┼───────────┤
│ Net Sales│ $50,000  │ $30,000  │   40%     │  ← 浅蓝背景
│   COGS   │ $20,000  │ $15,000  │   25%     │  ← 缩进
│Gross Profit│$30,000 │ $15,000  │   50%     │
└──────────┴──────────┴──────────┴───────────┘
```

**特点**：
- ✅ 表头深蓝色 + 白色文字
- ✅ 行标签浅蓝色
- ✅ 无边框
- ✅ 100 张批量生成
- ✅ 同时输出 PNG + HTML

---

## 共同功能

| 功能 | 说明 |
|------|------|
| **随机数据生成** | 随机生成财务数字（带 `$` 和逗号） |
| **随机行列数** | 3-6 列，8-15 行（可配置） |
| **随机词库** | 从内置词库随机选择列名/行名 |
| **缩进支持** | 行标签支持 0/1/2 级缩进 |
| **双输出** | 同时生成 PNG 图片 + HTML 文件 |
| **批量生成** | 一次性生成 100-1000 张 |
